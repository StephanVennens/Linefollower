## specifications
<br />
microcontroller:arduino leonardo
<br />
motors: 6V 50:1 metal DC motor
<br />
h-bridge: DRV8835
<br />
sensors: QTR-8A
<br />
batteries: 2x Tenpower INR18650-28HE 2800mAh - 10A
<br />
wireless communication: HC-05 module
<br />
distance sensor - motors: 12,5cm
<br />
weight:220g
<br />
speed: 0,44m/s
<br />
